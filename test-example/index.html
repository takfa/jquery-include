Client-side include

A jQuery plugin

This page should contain some included lists - the included content is indicated with grey dotted borders.

  • The includes are handled client-side.
  • Nested includes are supported.
  • A jQuery includeReady event is provided for post-processing scripts.

Note: When used in a file system Firefox 3.x will not load includes above the current page file context. E.G. <span src="../includes/nav.html"> will not work when you load from a file:// url.
